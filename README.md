# Telegram OCR Bot

Этот репозиторий содержит простой пример телеграм-бота, который получает фотографии и отправляет распознанный текст в ответ. Бот использует библиотеку `pytesseract` для извлечения текста из изображений.

## Требования

- Python 3.10+
- `python-telegram-bot` версии 20+
- `pillow`
- `pytesseract`
- Установленный Tesseract OCR

## Установка зависимостей

```bash
pip install python-telegram-bot==20.* pillow pytesseract
```

Для корректной работы необходимо установить Tesseract OCR. В Ubuntu это можно сделать так:

```bash
sudo apt-get install tesseract-ocr
```

## Запуск

Перед запуском задайте переменную окружения `TELEGRAM_TOKEN` со значением токена вашего бота. Затем выполните:

```bash
python bot.py
```

Теперь отправьте изображение в чат с ботом, и он вернет распознанный текст.
